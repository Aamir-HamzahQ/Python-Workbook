# Python-Workbook
# 🐍 Python Fundamentals Project — Data Technician Bootcamp

This repository showcases my Python learning journey during **Week 6 of the Data Technician Bootcamp**, where I built a series of small programs to strengthen my understanding of core programming concepts. These exercises focused on writing clean, logical Python code that interacts with users, performs calculations, and automates simple tasks.

---

## 🚀 Project Overview

Across multiple guided exercises, I developed programs that:

- Greet users and collect input  
- Perform arithmetic calculations  
- Make decisions using conditional logic  
- Automate repetitive tasks with loops  
- Build small interactive applications such as an ATM simulator (as described in the workbook: *“Print a main menu… ask the user different options… use conditional statements…”* )

These tasks helped build confidence in writing Python code that is both functional and user‑friendly.

---

## 🧠 Key Python Skills Demonstrated

### 🔤 **[Variables & Data Types](ca://s?q=Explain_Python_variables_and_data_types)**
Used variables to store user input, numeric values, and intermediate results. Practised type casting (e.g., converting strings to integers) to ensure calculations ran correctly.

### 🖨️ **[print() and input()](ca://s?q=How_do_print_and_input_work_in_Python)**
Created interactive programs such as greeting messages, temperature converters, and calculators.  
Example from the workbook: *“Write a Python program to calculate the sum of two numbers entered by the user.”* 

### 🔀 **[If Statements](ca://s?q=Learn_Python_if_statements)**
Implemented decision‑making logic for tasks like:

- Even/odd number checks  
- Prime number detection  
- Positive/negative/zero classification  
- FizzBuzz (from Day 2 Task 2: *“If a number is divisible by 3, print ‘fizz’…”* )

### 🔁 **[For Loops](ca://s?q=Python_for_loop_examples)**
Used to iterate through lists, generate sequences, and solve problems such as:

- Printing the first 10 even numbers  
- Finding max/min values in a list  
- Summing positive integers  

### 🔄 **[While Loops](ca://s?q=Python_while_loop_tutorial)**
Built interactive, menu‑driven programs — especially the ATM simulation — where the loop keeps the program running until the user chooses to exit.

### 🧮 **[Calculations & Logic Building](ca://s?q=Python_basic_calculations)**
Created programs for:

- Temperature conversion  
- Rectangle area calculation  
- Compound interest  
- Leap year detection  
- Sum of digits  

These exercises strengthened my ability to break down problems and translate them into step‑by‑step logic.
